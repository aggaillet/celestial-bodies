Interplanetary distance calculator

This page calculates the distance between two celestial bodies using ephemerides data provided by the Solar System Portal API of the Institut de mécanique céleste et de calcul des éphémérides (IMCCE).







Please insert names of celestial bodies as defined by the Working Group for Planetary System Nomenclature.
Examples include: Mars, Earth, Ceres, 501, etc.
